# Revisión jurídica previa a publicación

1. Confirmar licencia y autoría de cada pictograma.
2. Revisar compatibilidad de licencias entre proveedores.
3. Mantener las atribuciones.
4. Confirmar finalidad informativa, accesible, institucional y no comercial.
5. Documentar cualquier imagen PNG incorporada manualmente.
