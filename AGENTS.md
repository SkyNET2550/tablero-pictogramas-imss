# Instrucciones permanentes del proyecto

## Actualización y verificación del navegador

Después de cualquier cambio solicitado en HTML, CSS, JavaScript, datos, recursos visuales, configuración o servidor:

1. Guardar todos los archivos modificados.
2. Reiniciar el servidor local cuando el cambio afecte al servidor o a su configuración.
3. Recargar la aplicación abierta en el navegador lateral derecho o navegador integrado.
4. Si el navegador integrado no está disponible, comprobar la aplicación en:

   ```text
   http://127.0.0.1:4173/
   ```

5. Verificar visual y funcionalmente el elemento modificado.
6. Confirmar que no existan errores de carga, sintaxis, consola, imágenes o interacción.
7. No indicar que un cambio está terminado hasta haber actualizado y comprobado el navegador.

Esta regla debe aplicarse automáticamente en cada solicitud posterior relacionada con el proyecto.
